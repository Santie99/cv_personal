# Archivos a reemplazar o crear

Como este Sprint 01 inicia el proyecto desde cero, debes copiar todos los archivos y carpetas del ZIP dentro de la carpeta raíz de tu nuevo proyecto.

## Crear / copiar en raíz

- `.env.example`
- `.gitignore`
- `README.md`
- `package.json`
- `package-lock.json`
- `tsconfig.json`
- `next.config.ts`
- `next-env.d.ts`
- `postcss.config.mjs`

## Carpetas completas a crear / reemplazar

- `app/`
- `components/`
- `lib/`
- `types/`
- `public/`
- `docs/`

## Importante

No copies estas carpetas si aparecen en tu computador después de instalar o correr el proyecto:

- `node_modules/`
- `.next/`

Estas carpetas se generan automáticamente con `npm install` y `npm run dev/build`.
