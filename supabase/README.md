# Supabase · Sprint 02

1. Crea un proyecto en Supabase.
2. En SQL Editor ejecuta `schema.sql`.
3. Luego ejecuta `seed.sql`.
4. Copia `Project URL` y `anon public key` a `.env.local`.
5. Reinicia `npm run dev`.

Las políticas RLS permiten lectura pública del contenido publicado y escritura pública solo para `contact_messages`. La lectura de mensajes queda reservada para admins en fases posteriores.
